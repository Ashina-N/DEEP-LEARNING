Face Mask Detection is an artificial intelligence project that identifies whether a person is wearing a face mask or not using images or video.In this project, a dataset containing images of people with masks and without masks is used to train the model. Each image is labeled according to whether the person is wearing a mask.First, the system detects the face in the image or video. Then the face image is preprocessed (resized and normalized). After that, the CNN model learns patterns that distinguish masked faces from unmasked faces.When a new image or live webcam feed is given, the model analyzes the face and predicts whether the person is wearing a mask or not.
